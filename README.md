# 資安與數位平權 - 粒子互動測驗

這是一個結合 **MediaPipe Hands** 手勢辨識技術與 **HTML5 Canvas** 粒子系統的互動式測驗網頁。玩家可以透過手勢與網頁進行互動，在參與遊戲的過程中學習資產安全與數位平權的核心概念。

## 🚀 特色功能

- **手勢控制**：完全不需滑鼠或鍵盤，使用 1-4 根手指選擇答案，5 根手指重新開始。
- **粒子視覺特效**：15,000 個粒子組成的動態背景，受手勢引力影響。
- **響應式設計**：支援各類螢幕尺寸。
- **即時回饋**：提供正確答案的詳解，強化學習效果。

## 🎮 如何遊玩

1. 開啟網頁後，點擊「開始遊戲」。
2. 授予攝影機權限後，系統將進入 **5 秒倒數**。
3. 題目出現後，在攝影機前比出對應手指數量：
   - ☝️ **1 根手指**：選項 A
   - ✌️ **2 根手指**：選項 B
   - 🤟 **3 根手指**：選項 C
   - 🖖 **4 根手指**：選項 D
4. 保持手勢約 1 秒，待進度條填滿後即可送出。
5. 遊戲結束後，比出 🖐️ **5 根手指** 即可重新挑戰。

## 🛠️ 技術棧

- **MediaPipe Hands**：用於即時手勢偵測。
- **Canvas API**：高效能粒子物理運算。
- **Google Fonts (Orbitron, Noto Sans TC)**：現代化 UI 設計。

## 📂 專案結構

- `index.html`: 遊戲主要邏輯與介面。
- `help.html`: 遊戲介紹與操作指南。
- `questions.csv`: 題庫資料（可自行修改內容）。
- `gestures_guide.png`: 操作指南圖示。

## 🌐 部署

本專案可以直接上傳至 **GitHub Pages** 進行部署：
1. 將所有檔案上傳至 GitHub 儲存庫。
2. 在 Settings -> Pages 中選擇主要分支。
3. 儲存後即可獲得靜態網頁網址。
